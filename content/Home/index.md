# 1. Welcome to Obsurdian

This is your **system engineering documentation portal** built with Dash and DMC.

## 1.1 Features

- 📁 Recursive folder navigation
- 🎨 Dark/light theme toggle
- 📊 Auto-numbered headings
- 📋 Easy table syntax
- 🖼️ Easy image syntax

## 1.2 Getting Started

1. Navigate through folders on the left
2. Click any document to view
3. Toggle theme with the sun/moon icon
