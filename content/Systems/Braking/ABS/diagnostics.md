# 1. ABS Diagnostics

## 1.1 Overview

This section covers ABS (Anti-lock Braking System) diagnostics.

## 1.2 Diagnostic Codes

| Code | Meaning | Action |
|------|---------|--------|
| C0110 | Brake Fluid Level | Check reservoir |
| C0120 | Wheel Speed Sensor | Check sensor |
| C0130 | ABS Control Module | Check module |

## 1.3 Troubleshooting

1. Connect diagnostic tool
2. Read ABS codes
3. Follow action for each code
4. Clear codes after repair
